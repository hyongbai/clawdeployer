2

3
打开终端 → 龙虾已就绪

Demo 2 · 推推对话

1
打开推推 → 找到龙虾
2
直接对话 → 执行任务
3
查看结果 → 实时反馈

⏱ 3 分钟

Demo 3 · 龙虾教练训练专家虾

1
打开龙虾教练

3

⏱ 3 分钟
