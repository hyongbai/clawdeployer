# claw-pub-deploy

多项目静态部署仓库。通过 GitHub Pages 提供固定访问链接。

## 规则

- 每个部署项目占一个独立的顶层目录
- 根目录不放任何项目文件（只放本说明和通用配置）
- 目录名即访问路径：`https://hyongbai.github.io/claw-pub-deploy/<目录名>/`
- 各项目互相独立，互不影响

## 部署方式

将构建产物放入对应目录，push 到 `main` 分支即自动生效（GitHub Pages 从 main 分支根目录部署）。
